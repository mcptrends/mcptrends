# mcptrends

Adoption trends and momentum rankings for MCP (Model Context Protocol) servers.

The official MCP registry and GitHub tell you what exists. Nothing tells you what's growing. mcptrends tracks the MCP ecosystem over time and surfaces rising servers before they peak.

**Status: pre-release.** The `mcptrends` packages on [npm](https://www.npmjs.com/package/mcptrends) and [PyPI](https://pypi.org/project/mcptrends/) are name reservations for the upcoming CLI and SDK. Follow progress at [mcptrends.dev](https://mcptrends.dev).

## What's coming

- A public leaderboard of trending MCP servers
- A weekly trending digest

Rankings are built from public adoption signals tracked daily across the ecosystem.
